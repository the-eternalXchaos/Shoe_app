## Getting started

Prerequisites: Flutter SDK and a supported editor (VS Code, Android Studio).

1) Clone and open the project

```powershell
git clone https://github.com/the-eternalXchaos/shoe_app.git
cd "shoe_app"
```

2) Fetch packages and run

```powershell
flutter pub get
flutter run
```
